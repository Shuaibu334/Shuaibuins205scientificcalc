# Scientific Calculator

A modern Scientific Calculator developed using **HTML, CSS, and JavaScript**. The application performs both basic arithmetic and scientific calculations through a clean, responsive, and user-friendly interface.

 Features

 Basic Operations
- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Percentage (%)
- Decimal Numbers

 Scientific Functions
- Square Root (√)
- Square (x²)
- Cube (x³)
- Power (xʸ)
- Sine (sin)
- Cosine (cos)
- Tangent (tan)
- Logarithm (log)
- Natural Logarithm (ln)
- Factorial (x!)
- Pi (π)
- Euler's Number (e)

 Other Features
- Clear Screen
- Backspace
- Keyboard Support
- Responsive Design
- Modern User Interface

Technologies Used

- HTML5
- CSS3
- JavaScript 
- Docker

How to Run the Project

### Method 1

Open **ins205calc.html in any modern web browser.

Running with Docker
 Build the Docker Image

bash
docker build -t scientific-calculator .

Run the Docker Container

bash
docker run -d -p 8080:80 scientific-calculator


Open your browser and visit:

http://localhost:8080
